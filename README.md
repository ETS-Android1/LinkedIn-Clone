# LinkedIn-Clone

LinkedIn-clone is an android application with functionalities like one-tap login, add post, add a story, like a post, send connection request.

# Technology Used -
* XML
* Java
* Firebase

# Requirements -
*Android 5.0 and Above
*Min sdk version 21


# Permissions -
* Internet
